# 🧮 Responsive Keyboard Controlled Calculator

A modern and responsive calculator built using **HTML5**, **CSS3**, and **JavaScript**. This project supports both **keyboard input** and **button clicks**, performs mathematical operations through separate functions, validates user input, and provides a clean, responsive user interface.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Modulus
- ⌨️ Keyboard-controlled input
- 🖱️ Button click support
- 🚫 Invalid key validation with alert
- ⌫ Backspace support
- ⏎ Enter key to calculate result
- 🛡️ Divide-by-zero protection
- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface

---

## 🛠️ Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6)

---

## ✨ Functionalities

- Accepts numbers using the keyboard.
- Supports keyboard operators:
  - ➕ `+`
  - ➖ `-`
  - ✖️ `*`
  - ➗ `/`
  - 📊 `%`
- Prevents invalid keyboard input.
- Displays an alert when invalid keys are pressed.
- Supports Backspace key.
- Supports Enter key to calculate.
- Performs calculations using separate JavaScript functions.
- Prevents division by zero.
- Responsive layout for Desktop, Tablet, and Mobile devices.
